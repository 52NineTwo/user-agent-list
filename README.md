# user-agent-list

基于 [NetDiscovery](https://github.com/fengzhizi715/NetDiscovery) 爬虫框架抓取的常用浏览器的 user agent 以及 google、baidu 爬虫的 user agent

方便在其他项目中使用这些 user agent
